# Studio Nouveau

Art That Moves People - Three artists. One vision. Original paintings and creative works crafted with passion and precision in Baton Rouge, Louisiana.

## Tech Stack

- React + TypeScript
- Tailwind CSS
- GSAP
- Vite

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Deployed on Vercel with automatic deployments from the main branch.
